 INSTALLATION
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Open Timeprint.dmg (double-click)
2. Drag Timeprint.app to your Applications folder
3. Launch Timeprint from Applications
4. Grant Full Disk Access when prompted (required for Screen Time data)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 GRANTING FULL DISK ACCESS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Timeprint needs Full Disk Access to read Apple's Screen Time database. 
This is required for the app to function.

1. Open System Settings
2. Go to Privacy & Security → Full Disk Access
3. Click the + button
4. Navigate to Applications and select Timeprint
5. Restart Timeprint

Your data never leaves your device. Timeprint is 100% local and private.
