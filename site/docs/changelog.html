2.5.0 — Firefox, local snapshots, and MCP toggle

Date: May 15, 2026

  • Added Firefox support to Web History.
  • Added the automatic readable snapshot at ~/Library/Application Support/time.md/screen-time-snapshot.json.
  • Added live formatted auto-export as screen-time-auto.<ext> using your selected export format and sections.
  • Added a global MCP server on/off switch in Settings.
  • Changed Web History to read local browser databases directly for direct-distribution builds.
WEB HISTORY GUIDE EXPORT GUIDE CLI & MCP GUIDE

2.3.0 — Optional input tracking

Date: May 5, 2026

  • Added opt-in keystroke and cursor tracking for typing intensity, top typed words, top typed keys, and cursor heatmaps.
  • Added privacy guards: Secure Input redaction, default excluded apps, retention windows, and a global ⌥⌘P pause shortcut.
  • Added input-tracking tools to the bundled MCP server.
  • Moved input rows into input-tracking.db so high-volume mouse events do not bloat the main screen-time database.
INPUT TRACKING GUIDE

2.2.0 — Direct distribution and no subscription

Date: May 5, 2026

  • Moved time.md to direct distribution outside the Mac App Store.
  • Removed StoreKit subscription code from the app.
  • Kept local browser-history tracking available by distributing outside the App Store sandbox constraints.
  • Auto-updates are delivered through Sparkle.
Current checkout model:

The website now offers a free download, a card-backed 14-day trial from inside the app, and a one-time desktop license.

2.0.0 — Rebuilt macOS interface

Date: April 11, 2026

  • Added the current Overview, Review, Details, Projects, Rules, and Reports screens.
  • Added context-switch analysis, category/project rollups, weekday averages, and richer report data.
  • Reorganized the sidebar into Tracking, Organize, Data, and System groups.
  • Updated keyboard shortcuts to the current screen layout.
